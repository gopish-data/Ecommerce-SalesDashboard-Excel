# Ecommerce Sales Analysis Dashboard (Excel)

## About This Project
This is an Excel dashboard project created to analyze ecommerce sales data and understand overall business performance.

The goal of this project is to track important metrics like sales, profit, number of orders, and profit margin, and to identify trends across categories and regions.

## What I Did in This Project
- Cleaned and organized the dataset
- Used Pivot Tables for analysis
- Created calculated fields for KPIs
- Designed KPI cards for Sales, Profit, Quantity, Orders and Profit Margin
- Built different charts like:
  - Monthly Sales & Profit chart
  - Category-wise Profit analysis
  - Sales share percentage (Donut chart)
  - Sales by State (Map)
  - Top 5 Sub-Categories
- Added interactive filters (Year, Segment, Region)

## Key Metrics Tracked
- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Profit Margin %
- Year-over-Year Growth %

## Insights Observed
- Some categories contribute more to overall profit
- Sales performance varies by region
- Certain sub-categories generate higher revenue
- Profit margin changes based on sales trends
